  • m = Maskulinum, f = Femininum, n = Neutrum, p = Plural

  • N = Nominativ, A = Akkusativ

  • b = bestimmt, u = unbestimmt

Nominativ Singular = (m, f, n)

bestimmt = b

Ist das (der Apfel, die Zwiebel, das Brot)?
Is this the apple?

  • Ja. Das ist (der Apfel, die Zwiebel, das Brot).
    Yes. This is the apple.

  • Nein. Das ist nicht (der Apfel, die Zwiebel, das Brot). Das ist die Melone!
    No. This is not the apple. This is the melon!

unbestimmt = u

Ist das (ein Apfel, eine Zwiebel, ein Brot)?
Is this an apple?

  • Ja. Das ist (ein Apfel, eine Zwiebel, ein Brot).
    Yes. This is an apple.

  • Nein. Das ist (kein Apfel, keine Zwiebel, kein Brot). Das ist eine Melone!
    No. This is not an apple. This is a melon!

Nominativ Plural = p

bestimmt = b

Sind das (die Äpfel, die Zwiebeln, die Brote)?
Are these the apples?

  • Ja. Das sind (die Äpfel, die Zwiebeln, die Brote).
    Yes. These are the apples.

  • Nein. Das sind nicht (die Äpfel, die Zwiebeln, die Brote). Das sind die Melonen!
    No. These are not the apples. These are the melons!

unbestimmt = u

Sind das (Äpfel, Zwiebeln, Brote)?
Are these apples?

  • Ja. Das sind (Äpfel, Zwiebeln, Brote).
    Yes. These are apples.

  • Nein. Das sind (keine Äpfel, keine Zwiebeln, keine Brote). Das sind Melonen!
    No. These are not apples. These are melons!

Akkusativ Singular = A m

Akkusativ = Nominativ außer bei m.

bestimmt = b

Siehst du den Apfel?
Do you see the apple?

  • Ja. Ich sehe den Apfel.
    Yes. I see the apple.

  • Nein. Ich sehe nicht den Apfel. Es ist zu dunkel.
    No. I do not see the apple. It is too dark.

unbestimmt = u

Siehst du einen Apfel?
Do you see an apple?

  • Ja. Ich sehe einen Apfel.
    Yes. I see an apple.

  • Nein. Ich sehe keinen Apfel. Aber ich sehe *einen Fernseher.
    No. I do not see any apple. But I see a TV.
